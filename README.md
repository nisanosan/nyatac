# ***N***ope. (It's) ***Y***et ***A***nother ***T***utorial ***A***dhering ***C***ompiler (N.YATAC)
Pretty much what the title/project name says! It's yet another compiler project (made by following a tutorial) on the internet with ***no intention of actual usage and maintenance*** soo if you were looking for some sort of compiler and you ended up here, I'm sorry ***this is not the compiler you are looking for***.

This so called tutorial hell project was made by following [Bisqwit's](https://www.youtube.com/@Bisqwit) ["How to Create a Compiler"](https://youtube.com/playlist?list=PLzLzYGEbdY5n9ITKUqOuRjXkRU5tMW2Sd) tutorial series.

This is written in ***C++*** so if you *"love"* the syntactic sugar of C++ (*which clearly makes you feel like a real programmer*) then... Okay IDK *what* you would do exactly with a *toy project made by following a freely available tutorial* but... Hey it's FOSS so it's your wish! (Licensed under [MIT License](./LICENSE.txt))
